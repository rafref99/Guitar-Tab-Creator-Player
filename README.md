## 📖 What is a Tab?

**Tab** is short for **Tablature**. It is a notation system that describes how a song or melody is played on a guitar.

If you've never read guitar tabs before, don't worry! There are many great tutorials on YouTube, and the basics are quite easy to learn.

# ✨ Instructions

## 📂 Loading Existing Songs

1. Click **New**.
2. Select **Browse Tabs**.
3. Click any song title.

The selected song will be loaded instantly.

---

## ▶️ Play Controls

Next to the **Capo**, **Volume**, and **BPM** sliders, you'll find the **Play** controls.

### Available Buttons

* **Play Current Stave** – Repeats only the currently selected stave.
* **Play Entire Song** – Plays the whole song once.
* **Stop** – Stops all playback immediately.

---

## 🎚️ Sliders

### Capo

If you're unfamiliar with a capo, a quick Google search will explain it better than a paragraph here!

### Volume

Controls playback volume.

### BPM

Controls how fast the song is played.

You can also place BPM markers directly into the tab to change speed during playback.

Example:

```text
{♩=100}
```

This allows songs to start slowly and speed up later, or vice versa.

---

## 🎸 Frets

Most guitars have:

* 6 strings
* Multiple frets along the neck

The horizontal metal bars on the neck are called **frets**.

This project includes the first **12 frets**, which are the most commonly used.

### Adding Notes

1. Select a fret.
2. Click on one of the six tab lines.

Each line represents one guitar string.

---

## ⚙️ Modes

There are **4 editing modes**.

### 1️⃣ Add Mode

Used to add:

* Notes
* BPM markers

### 2️⃣ Move Mode

Used to move:

* Notes
* BPM markers

Movement is limited to **left and right** only.

### 3️⃣ Delete Mode

Used to delete:

* Notes
* BPM markers
* Group markers

### 4️⃣ Group Mode

Used to create navigation groups within a song.

Groups help organize different sections such as:

* Intro
* Verse
* Chorus
* Solo

---

## 📑 Groups

Groups are optional but highly recommended for larger songs.

### Features

* Quickly jump to any group using the buttons at the bottom of the screen.
* Organize sections of your song.
* Easily find important parts later.

### Deleting Groups

1. Select **Delete Mode**.
2. Click the group's numbered button.

---

## 🎼 Staves

A **stave** is the page where music is written.

Traditional music notation uses 5 lines, but guitar tablature uses **6 lines**, representing the guitar's six strings.

### Managing Staves

#### Add a Stave

Click:

```text
➕
```

#### Remove a Stave

Click:

```text
➖
```

#### Navigate Between Staves

Use:

```text
⪡   ⪢
```

or the keyboard arrow keys.

### Important

Removing a stave does **not** delete the notes stored on it. Your data remains saved and can be accessed again later.

---

## ⭐ Tips

* Add BPM markers to create tempo changes.
* Use the Tab Browser to discover songs created by other users.
* Save your song code regularly.
